# RegemX Official Website Repository / RegemX 官方网站仓库  
RegemX — A RuleGems-based Minecraft multiplayer server / RegemX — 基于 RuleGems 的 Minecraft 多人服务器  

This repository hosts the full source code and resources for the RegemX official website, automatically deployed via GitHub Pages.  
Admins can directly maintain announcements, events, download links, and documentation here; changes go live on commit.  
本仓库托管 RegemX 官方网站的全部源码与资源，采用 GitHub Pages 自动发布。  
管理员可直接在此维护公告、活动、下载链接及文档，提交即上线。

## Site Overview / 站点概况
- Framework / 框架：Native HTML + GitHub Pages  
- Domain / 域名：regemx.github.io (already bound to regemx.org)（已绑定 regemx.org）  
- Main directories / 主要目录：  
  ├─ `index.html` Homepage / 首页  
  └─ `images/` Images & static assets / 图片与静态资源
